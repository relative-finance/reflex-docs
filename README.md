# Reflex Docs

User-facing documentation for the Reflex trading terminal ([app.reflex.trade](https://app.reflex.trade)).

Hosted at: [docs.reflex.trade](https://docs.reflex.trade)

## Local development

- `npm install`
- `npm start`

## Build

- `npm run build`

## Deployment

Auto-deploys to Vercel on every push to `main`.

Custom domain: `docs.reflex.trade`
