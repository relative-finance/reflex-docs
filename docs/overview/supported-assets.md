# Supported Assets

Reflex mirrors Hyperliquid markets in near real time.

## Market Types

- Perpetual futures
- Spot markets
- HIP-3 markets (where available in your environment)

## How to Check What Is Tradeable Right Now

1. Open the market selector in the terminal
2. Search by ticker (for example `BTC`, `ETH`, `SOL`)
3. Confirm the market type before submitting orders

## Notes

- Available symbols can change as Hyperliquid listings change
- One-click feed trading currently routes to perp markets
- If an AI-extracted ticker is not tradeable, Reflex falls back to a valid default in the UI
