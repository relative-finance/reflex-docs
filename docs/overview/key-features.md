# Key Features

## Live Today

| Area | What you get |
|---|---|
| Trading terminal | Market, Limit, TWAP, Scale, trigger orders, TP/SL, Reduce Only, leverage control |
| News feed | X, Telegram, and curated RSS/news in one stream |
| AI context | Sentiment and ticker extraction on feed items |
| One-click execution | Long/Short directly from feed cards using your active preset |
| Quick trade sizes | Pulse mode shortcut buttons (preset-defined sizes) |
| Layout modes | Classic and Pulse layouts with drag/resize/hide controls |
| Data tables | Balances, Positions, Open Orders, TWAP, Funding History, Order History, Trade History |
| Bridge | Built-in LI.FI bridge modal from the header |
| Telegram integration | Account linking, notifications, and trading conversation flow |
| Trade Ideas | "Suggest Trade Idea" action from news cards with history panel |
