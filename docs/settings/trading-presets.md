# Settings: Trading Presets

`Settings -> Preset` is the main control center for creating and editing presets.

## What You Can Do

- Create/edit up to 10 presets
- Set size, leverage, slippage, TP/SL, mode, and order-type behavior
- Configure quick trade shortcut sizes used in Pulse feed cards

## Save Behavior

Preset changes are persisted per wallet/account context and used immediately by feed trading actions.
