# Onboarding Flow

After first login, Reflex runs a 6-step onboarding flow.

## Step 1: Welcome

Quick intro and start button.

## Step 2: Create Account

Reflex runs account setup actions and signatures:
- Hyperliquid legal/account checks
- Reflex authentication token setup
- Agent wallet setup
- Builder fee authorization

If funds are insufficient for agent wallet setup, the flow surfaces the Bridge shortcut.

## Step 3: Choose Default Mode

Select your default layout style:
- Pulse (signal-first)
- Classic (chart/execution-first)

## Step 4: Setup News Sources

Add your X and Telegram subscriptions.

## Step 5: Create Trading Presets

Configure and save your trading presets (used by one-click feed trades).

## Step 6: Setup Notifications

Configure delivery channels (push/Telegram) and feed alert preferences.

## Skip / Resume

You can skip individual onboarding steps and complete configuration later in Settings.
