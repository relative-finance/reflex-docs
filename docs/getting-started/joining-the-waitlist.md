# Joining the Waitlist

Reflex is currently access-gated.

## Steps

1. Go to `https://reflex.trade`
2. Submit your waitlist request
3. Wait for approval
4. Open `https://app.reflex.trade/trade`

If your account is newly approved, the terminal may ask for an invite code during first-time connect.

## After Approval

On first successful sign-in, Reflex launches the guided onboarding flow automatically.

## FAQ

**Can I use multiple wallets?**
Yes, but access is wallet-specific.

**How long does approval take?**
Approvals are batch-based and can vary.
