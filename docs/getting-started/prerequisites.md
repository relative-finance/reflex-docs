# Prerequisites

Before you start trading on Reflex, make sure you have:

## 1. An Ethereum Wallet or Email Address

A wallet that can connect and sign messages/transactions (for example MetaMask, Coinbase Wallet, Rainbow, or WalletConnect-compatible wallets). If you do not have a wallet, then you can sign up using your email and Privy will create a linked wallet for you.

## 2. Access Approval

Reflex currently uses access gating.

- Join the waitlist on `reflex.trade`
- Obtain an invite code 
- Navigate to the terminal, trigger the connect wallet flow and use your invite code when prompted

## 3. Hyperliquid Funding

You need funds on the Hyperliquid L1 to trade.

- Recommended: use Reflex's built-in Bridge (powered by https://li.fi)
- Alternative: deposit directly via Hyperliquid

Important:
- You need to have at least $1 deposited in your Hyperliquid account to initialize agent-wallet permissions during onboarding
- Trading requires available margin/collateral on Hyperliquid

## 4. A Desktop Browser

Reflex is optimized for desktop web usage.

## 5. Optional: Telegram Account

If you want Telegram notifications or Telegram trading, have Telegram ready so you can complete the link flow during or after onboarding.
