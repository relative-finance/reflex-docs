# Bridging Funds to Hyperliquid

Reflex includes a built-in LI.FI bridge modal.

## Open the Bridge

Use the `Bridge` button in the terminal header.

## What You Can Configure

- Source chain
- Destination chain (typically Hyperliquid path)
- Source token
- Destination token
- Amount

Chains/tokens are fetched dynamically from LI.FI support.

## Execution Flow

1. Select networks/tokens
2. Review quote, fees, and ETA
3. Approve token if required
4. Sign and submit bridge transaction
5. Track completion status in the modal

## Cost Components

- Source-chain gas
- Bridge route/protocol fees (included in quote)
- No additional fees charged by Reflex to bridge
