# Setting Up Alerts

Use the Notifications panel (bell icon in header) or onboarding step 6.

## Setup Steps

1. Enable one or both channels (`Push`, `Telegram`)
2. Link Telegram if needed
3. Enable feed alerts
4. Choose delivery channel(s) for feed alerts
5. Save preferences

## Optional Source Granularity

You can also toggle alerting for individual followed accounts/channels within the same preferences flow.
