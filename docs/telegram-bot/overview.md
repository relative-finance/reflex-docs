# Telegram Bot Overview

The Telegram integration supports account linking, notifications, and conversational trade execution.

## Core Capabilities

- Link terminal account to Telegram
- Receive Reflex notifications in Telegram
- Submit trades through guided Telegram flow
- Query request status

## Current Command Surface

- `/start` (or `/help`)
- `/trade`
- `/wallet`
- `/status`

## Important

Telegram trading uses your authorized agent/API wallet permissions, not direct custody of your main wallet funds.
