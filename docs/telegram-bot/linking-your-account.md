# Linking Your Account

Linking is done with a one-time code from Reflex.

## Steps

1. Open Telegram settings flow in Reflex (`Telegram` page or Notifications onboarding)
2. Click `Open Telegram Bot`
3. Reflex generates a one-time code
4. Send/enter that code in the bot chat
5. Reflex auto-detects success and marks Telegram as linked

## If Linking Fails

- Regenerate a fresh code
- Confirm you are using the same Reflex account/wallet session
- Retry after checking Telegram connectivity
