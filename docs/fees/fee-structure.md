# Fee Structure

Trading through Reflex includes:

1. Hyperliquid trading fees
2. Reflex builder fee (where applicable for routed order flow)

## Reflex Builder Fee

Current web terminal builder fee:
- Web terminal: `0.01%`

Builder fee approval is a one-time authorization flow and can be revoked from Hyperliquid settings.

## Hyperliquid Fees

Hyperliquid applies its own maker/taker and tiered volume fee logic.

For the most accurate current values:
- Check in-app fee schedule/modal
- Check official Hyperliquid docs

## Important

Total trade cost depends on order type, maker/taker outcome, volume tier, and staking/discount context.
