# Hyperliquid Trading Fees

Reflex executes on Hyperliquid, so Hyperliquid fee logic always applies.

## Fee Inputs That Matter

- Maker vs taker execution
- Rolling volume tier
- Market type (perp/spot)
- Any Hyperliquid-native discount programs

## Where to Check Live Values

- Hyperliquid official documentation
- Reflex in-app fee schedule UI (when available in your view)

Official docs: [Hyperliquid trading fees](https://hyperliquid.gitbook.io/hyperliquid-docs/trading/fees)
