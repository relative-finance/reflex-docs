# Wallet Security

## Core Model

Reflex is non-custodial.

- Funds remain on Hyperliquid
- Reflex does not ask for seed phrases
- Authentication uses SIWE signatures

## Safe Usage Checklist

- Bookmark official Reflex URLs
- Verify every signature prompt context
- Use a dedicated trading wallet where practical
- Prefer hardware-wallet-backed custody for large balances
- Disconnect if session behavior looks suspicious

## Phishing Warnings

Reflex should never ask you to share:
- Seed phrase
- Private key
- Direct fund-transfer approval for "support" reasons

Treat unsolicited DMs claiming to be Reflex support as suspicious until verified through official channels.
