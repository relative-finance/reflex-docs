# Terminal Modes

Reflex has two main layout modes.

## Pulse Mode

Signal-first layout.

Typical structure:
- Left side: chart + data tables
- Right side: feed panel spanning height
- Optional trade-ideas panel (can be expanded by clicking the + icon on any open panel)

Use Pulse if you trade from incoming signals and want feed context always visible.

## Classic Mode

Execution-first layout.

Typical structure includes:
- Chart
- Feed
- Order Book panel
- Data Tables
- Account Overview
- Optional Polymarket and Trade Ideas panels

Use Classic if you prefer a more traditional terminal arrangement.

## Switching Modes

Use the `Classic / Pulse` toggle in the header.

Mode preference is persisted locally, and each mode keeps its own layout context.
