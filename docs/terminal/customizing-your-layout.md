# Customizing Your Layout

You can customize both panel visibility and arrangement.

## What You Can Change

- Drag panel position
- Resize panel dimensions
- Hide/show supported panels
- Choose which feed/order-book tabs are visible

## Layout Settings Workflow

1. Open `Settings -> Layout`
2. Toggle widgets/panels on or off
3. Close/save

## Resetting

Use either:
- Header `Reset Layout` button, or
- `Reset` inside Layout settings

Reset restores default panel visibility/positions for the active mode.

## Mode-Specific Behavior

Classic and Pulse layouts are managed independently, so changes in one mode do not overwrite the other mode's panel arrangement.
